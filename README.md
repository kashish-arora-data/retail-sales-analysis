# Retail Sales Analysis

## Project Overview

This project analyzes retail sales data using SQL and Power BI to identify sales trends, customer behavior, and regional performance.

## Dataset

The dataset contains retail order information including:

- Order ID
- Order Date
- Customer Name
- Segment
- City
- State
- Region
- Category
- Sub-Category
- Product Name
- Sales

## Business Questions

1. Which regions generate the highest sales?
2. Which cities contribute the most revenue?
3. Which customer segments are most valuable?
4. Which products and categories perform best?
5. Are sales concentrated among a small number of customers?
